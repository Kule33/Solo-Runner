**SELF TRAINNING RUNNING TRACK SYSTEM**

Developed a hardware and web-based system to support athletes in independent training by providing accurate timing, instant feedback, and video/picture analysis. Implemented precise start and end detection using LDR sensors and limit switch modules. Integrated ESP32 modules with Wi-Fi to enable wireless data transfer and a Firebase-based web interface for real-time storage and visualization of timing data and recorded videos.
Impact: Enabled precise performance tracking, immediate feedback, and improved training efficiency for athletes.
Technologies & Components: ESP32, LDR sensors, camera modules, display modules, Firebase, pocket Wi-Fi, sound counter
